# 接口文档

## 基本信息

- **Base URL**: `/api/v1`
- **数据格式**: JSON
- **编码**: UTF-8

---

## 通用响应格式

```json
{
  "code": 200,
  "data": { ... }
}
```

失败时 `code` 不为 200，`data` 可能为错误信息或具体业务数据。

---

## 健康检查

### GET /

健康检查接口。

**响应示例**:
```json
{
  "status": "ok"
}
```

## 原始交割单

### GET /api/v1/deliveries

分页查询原始交割单。

**查询参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认 1 |
| page_size | int | 否 | 每页数量，默认 10，最大 100 |
| stock_code | string | 否 | 股票代码 |
| stock_name | string | 否 | 股票名称 |
| trade_type | string | 否 | 交易类别（买入/卖出） |
| trade_date_start | date | 否 | 成交日期开始 (YYYY-MM-DD) |
| trade_date_end | date | 否 | 成交日期结束 (YYYY-MM-DD) |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "total": 100,
    "page": 1,
    "page_size": 10,
    "items": [
      {
        "id": 1,
        "trade_date": "2024-01-15",
        "trade_time": "09:30:00",
        "stock_code": "600000",
        "stock_name": "浦发银行",
        "trade_type": "买入",
        "quantity": 1000,
        "trade_price": "10.50",
        "occur_amount": "10500.00",
        "deal_amount": "10500.00",
        "fee": "30.00",
        "remark": null,
        "created_at": "2024-01-15T09:30:00"
      }
    ]
  }
}
```

### POST /api/v1/deliveries

创建原始交割单。

**请求体**:
```json
{
  "trade_date": "2024-01-15",
  "trade_time": "09:30:00",
  "stock_code": "600000",
  "stock_name": "浦发银行",
  "trade_type": "买入",
  "quantity": 1000,
  "trade_price": 10.50,
  "occur_amount": 10500.00,
  "deal_amount": 10500.00,
  "fee": 30.00,
  "remark": null
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| trade_date | date | 是 | 成交日期 |
| trade_time | time | 否 | 成交时间 |
| stock_code | string | 是 | 股票代码 |
| stock_name | string | 是 | 股票名称 |
| trade_type | string | 是 | 交易类别（买入/卖出） |
| quantity | int | 是 | 数量 |
| trade_price | decimal | 是 | 成交价格 |
| occur_amount | decimal | 是 | 发生金额 |
| deal_amount | decimal | 是 | 成交金额 |
| fee | decimal | 否 | 手续费，默认 0 |
| remark | string | 否 | 备注 |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "id": 1,
    "trade_date": "2024-01-15",
    "trade_time": "09:30:00",
    "stock_code": "600000",
    "stock_name": "浦发银行",
    "trade_type": "买入",
    "quantity": 1000,
    "trade_price": "10.50",
    "occur_amount": "10500.00",
    "deal_amount": "10500.00",
    "fee": "30.00",
    "remark": null,
    "created_at": "2024-01-15T09:30:00"
  }
}
```

## 已清仓股票统计

### GET /api/v1/cleared-positions

分页查询所有已清仓的买卖周期。

**查询参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| page | int | 否 | 页码，默认 1 |
| page_size | int | 否 | 每页条数，默认 20，最大 100 |
| stock_code | string | 否 | 股票代码模糊搜索 |
| profit_filter | string | 否 | 盈亏筛选：all/profit/loss，默认 all |
| start_date | date | 否 | 建仓日期起 (YYYY-MM-DD) |
| end_date | date | 否 | 建仓日期止 (YYYY-MM-DD) |
| order_by | string | 否 | 排序字段：close_date/profit_loss/profit_rate/hold_days/open_date，默认 close_date |
| order_dir | string | 否 | 排序方向：asc/desc，默认 desc |

**响应示例**:
```json

  "data": {
    "total": 50,
    "page": 1,
    "page_size": 20,
    "list": [
      {
        "id": 1,

        "cycle_index": 1,
        "open_date": "2024-01-10",
        "close_date": "2024-01-20",
        "hold_days": 10,
        "total_buy_amount": "100000.00",
        "total_sell_amount": "105000.00",
        "total_buy_qty": 10000,
        "profit_loss": "5000.00",
        "profit_rate": "5.00",
        "avg_buy_price": "10.00",
        "avg_sell_price": "10.50",
        "record_ids": "1,2,3",
        "notes": null,
        "updated_at": "2024-01-20T15:00:00",
        "cost_vs_open": null,
        "cost_vs_close": null
      }
    ],
    "last_clean_time": "2024-01-20T15:00:00"
  }
}
```

### POST /api/v1/cleared-positions/clean

清洗已清仓周期数据到预计算表。

**请求体**:
```json
{
  "start_date": "2024-01-01",
  "end_date": "2024-12-31",
  "stock_codes": ["600000", "600001"]
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| start_date | date | 否 | 开始日期 |
| end_date | date | 否 | 结束日期 |
| stock_codes | array | 否 | 股票代码列表，为空则清洗所有 |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "success": true,
    "cleaned_count": 10,
    "message": "清洗完成"
  }
}
```

### POST /api/v1/cleared-positions/notes

更新已清仓周期的交易笔记。

**请求体**:
```json
{
  "stock_code": "600000",
  "open_date": "2024-01-10",
  "close_date": "2024-01-20",
  "notes": "这只股票在1月10日建仓，1月20日清仓，盈利5%"
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| stock_code | string | 是 | 股票代码 |
| open_date | date | 是 | 建仓日期 |
| close_date | date | 是 | 清仓日期 |
| notes | string | 否 | 交易笔记 |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "success": true,
    "message": "笔记更新成功"
  }
}
```

## 行情数据同步

### POST /api/v1/quotes/sync

同步所有股票的行情数据。从每只股票最早交易日期开始，抓取每日行情数据。

**查询参数**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| force | bool | 否 | 是否强制更新已有数据，默认 false |

**响应示例**:
```json
{
  "code": 200,
  "data": {
    "message": "行情同步完成",
    "total_stocks": 10,
    "synced_count": 100
  }
}
```
